<Info>
  This process requires 2 things:

  1. A [Discord](https://support.discord.com/hc/en-us/articles/204849977-How-do-I-create-a-server) server

  2. <Tooltip tip="Administrator permissions grant full control over a Discord server.">Administrator</Tooltip> permissions in the Discord server
</Info>



<Steps>
  <Step title="Invite">
    Ensure you have **Administrator** permissions in the Discord that you're wanting to setup ERM in. Then, [invite the bot.](https://discord.com/oauth2/authorize?client_id=978662093408591912\&permissions=8\&scope=bot%20applications.commands)
  </Step>

  <Step title="Setup">
    Once you have invited the bot, you can then run the `/setup` command.
  </Step>

  <Step title="Basic Settings">
    Heres where you will setup the roles and bot prefix.  &#x20;
  </Step>
</Steps>
